# @sherpa/mcp

Sherpa's MCP server package.

## Install

```bash
npm install @sherpa/mcp
```

## Run

```bash
sherpa-mcp --agent-id main
sherpa-mcp-http --agent-id main --host 127.0.0.1 --port 8787
```

`@sherpa/mcp` exposes Sherpa over MCP stdio and streamable HTTP transports for local tool integration.
